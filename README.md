# TelegramBot <b>PDF>>TO>>MP3</b>
# EN
Telegram bot for reading text from PDF files

How does it work:
- Enter the language for voiceover('ru' or 'en')
- Drop PDF file up to 20MB!
- Get ready MP3 file with text from your PDF
